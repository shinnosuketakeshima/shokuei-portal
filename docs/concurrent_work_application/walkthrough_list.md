# 兼務届け申請一覧機能の実装と日付指定の改善

兼務届け申請に関する一覧画面と、特定の数日を申請する機能の実装が完了しました。

## 追加された機能

### 1. 申請一覧画面の追加
- ダッシュボードの左側メニューに新しく **「申請一覧」** を追加しました。
- これまで（テストを含め）提出されたすべての申請データを取得し、以下のように一覧表示します。
  - **申請日**（自動取得）
  - **申請教員**
  - **兼務先**
  - **期間 / 日付**

### 2. 「特定の数日」での申請対応
- 申請フォームの期間指定部分に **「期間の指定方法」** という選択肢を追加しました。
- **「特定の日付で指定（複数日など）」** を選ぶと、開始日・終了日の入力欄が消え、代わりに「令和6年5月10日、5月17日」のように**自由にテキスト入力できる欄**が出現します。
- これを使って提出すると、一覧画面にはそのテキストがそのまま表示されます。
- ダウンロードされるWordファイルには、以下のようになります。
  - 左側（開始日）：自由入力したテキスト（例：「令和6年5月10日、5月17日」）
  - 右側（終了日）：「（同左）」

## 確認事項
一覧画面やフォームの操作感をお試しいただき、特定の数日での運用がスムーズに行えるかご確認ください。
必要に応じてテキスト入力欄の名称変更や、Word側の表示ルールをさらに調整することも可能です。
