- ダッシュボードに **「学外兼務 承認申請（一般）」** のボタンを追加しました。

## ⚠️ 次のステップ：Wordファイルへのタグ埋め込み

Wordの「27.学外兼務承認申請書.docx」を開き、それぞれの入力欄に以下の【 】で囲まれたタグをご記入の上保存し、そのファイルを本番のシステムにアップロード（または上書き）してください。

### 📌 タグの一覧
*   右上の提出日: `【reiwaDate】`
*   所属: `【department】`
*   職名: `【title】`
*   氏名: `【name】`
*   1. 兼業する名称、所在地: `【workplace】`
*   2. 業務内容: `【jobDescription】`
*   3. 月額収入見込: `【income】`
*   4. 業務期間（開始）: `【periodStart】`
*   4. 業務期間（終了）: `【periodEnd】`
*   4. 業務日時（曜日・時間等）: `【dayAndTime】`
*   5. 当該業務と本学における…: `【relationComment】`
*   6. 本学の勤務・授業等の…: `【noHinderanceComment】`

※ 特定の日付での申請があった場合は、「非常勤用」の際と同じく `【periodStart】` の場所にテキストが入り、 `【periodEnd】` には（同左）が入る仕組みになっています。

タグの埋め込みが終わりましたら、Webサイトからテスト送信とダウンロードをお試しください！
